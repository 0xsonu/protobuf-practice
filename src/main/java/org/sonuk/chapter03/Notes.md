## Data Types

### Scaler Types
Basic Building Block in Proto by which we can build all our proto messages and using those messages we can
further create complex messages.


| Java Types | Proto Types  |
|------------|--------------|
| int        | int32/sint32 |
| long       | int64/sint64 |
| float      | float        |
| double     | double       |
| boolean    | bool         |
| String     | string       |
| byte[]     | bytes        |
